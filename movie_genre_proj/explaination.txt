This project uses a supervised machine learning model to categorise genres of movies in terms of their plot summaries. The data will be composed of movie descriptions and the corresponding genre description. As machine learning algorithms cannot work with raw text, the textual information is converted to numerical features by the use of the Term Frequency Inverse Document Frequency (TF-IDF) technique first. The TF-IDF obtains a movie description in a form of a vector by quantifying the significance of the words in comparison to the general sample, thus lessening the influence of frequently used words, which are not significantly enlightening.

The classification algorithms after the feature extraction include Support Vector Machine (SVM), Logistic Regression, and Naive Bayes to learn the patterns between the textual features and the genre labels. The model is trained using labelled training data and tested on a different test dataset with accuracy and classification measures. SVM is one of the applied classifiers that tend to have good performance in classification of text. The ultimate system can correctly forecast the genre of a new movie, solely on its plot description, thus it is able to satisfy the goal of making automated genre classification of movies using natural language processing features.
