The objective of this project will be to develop a machine learning model that can be used to identify fraudulent credit card transactions. The data has transaction-level characteristics and a binary target variable which shows the validity (0) or fraud (1) of a transaction. Because machine learning models need numerical features as their input, numerical features were used to train the model.

An implementation of Random Forest classifier was done to identify transactions. Random Forest is an ensemble learning algorithm, a set of decision trees combined together to enhance the precision and minimize overfitting. The training data was used in the model and the accuracy, confusion matrix, precision, recall, and F1-score were used to evaluate the model.

Despite the high overall accuracy of the model since the count of legitimate transactions was large, the recall of fraudulent transactions was rather small since the dataset is highly imbalanced. This shows how critical the issue of class imbalance is when it comes to recognizing fraud issues. On the whole, the model effectively proves that machine learning can be employed to identify fraudulent financial transactions.
